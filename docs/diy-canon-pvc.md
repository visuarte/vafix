# Apéndice práctico — Cañones de aire para FX

## Nivel básico: PVC manual (3-4 bares)

Construcción de un cañón de aire comprimido para simular explosiones
sin pólvora ni fuego. Útil para pruebas de concepto, ensayos de timing,
o producciones de bajo presupuesto.

## Materiales (ferretería España)

| Componente | Especificación |
|---|---|
| Tubería PVC presión | 75-110mm diámetro, para agua (no evacuación) |
| Válvula de bola | 1-2 pulgadas, apertura manual rápida |
| Válvula de neumático | De coche o bicicleta, para inflar |
| Pegamento PVC + teflón | Sellado hermético |

## Materiales de dispersión (inofensivos)

| Material | Efecto visual |
|---|---|
| Turba/tierra seca | Tierra negra de explosión |
| Corcho blanco triturado | Escombros, piedras falsas |
| Polvo de cacao / carbón vegetal | Hollín, color grisáceo |
| Maicena o talco | Nube de humo denso |
| Espuma/esponja pintada | Rocas grandes que rebotan |

## Seguridad

- Presión máxima: 3-4 bares (40-60 PSI)
- Nunca apuntar a personas
- Protección auditiva para el equipo
- Ensayar sin materiales antes del rodaje

## Nivel intermedio: acero + RF (6-8 bares)

| Componente | Especificación |
|---|---|
| Tanque | Extintor 6-9kg vacío (acero, 15+ bares) |
| Tubería | Hierro galvanizado o latón 1-2" |
| Válvula | Electroválvula NC 12V/24V, alto caudal |
| Batería | 12V moto o SAI |
| Mando | RF 12V (estilo puerta garaje) |
| Seguridad | Interruptor de llave en circuito |

**Mejoras respecto al PVC:**
- 2× más presión (8 bares vs 4)
- Disparo remoto desde 10m
- Sincronización exacta (electroválvula en milisegundos)
- Tanque metálico homologado (no se agrieta)

**Consumibles intermedios:**
- Tierra de Fuller (arcilla) en vez de maicena (no inflamable)
- Talco industrial gris para polvo de hormigón

## Para qué sirve en VAFIX

No para el rodaje real (Santiago usará equipos profesionales).
Sí para **validar la timeline**: construye el cañón, mide tiempos reales
entre apertura de válvula y dispersión, y ajusta la sincronización
antes de meter propane cannons de verdad.

El salto a electroválvula + RF permite sincronizar con código de tiempo,
que es exactamente lo que la timeline de VAFIX va a coordinar.
