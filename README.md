# VAFIX — Visual Assistant for FX

Planificación de efectos visuales físicos (F/x) para rodaje.
Proyecto actual: **StuG III vs T-34** (Santiago Espin).

## Qué es

Un tablero de planificación donde las 3 capas de una producción FX 
se ven juntas: **F/x (físicos)**, **SVfx (visuales)** e **Iluminación**.

Sin frameworks. Sin servidor. Sin internet. Funciona desde un archivo.

## Cómo usarlo

Abre `index.html` en cualquier navegador. No necesita instalación.
